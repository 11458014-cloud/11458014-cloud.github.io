# 🐍 貪吃蛇遊戲

一個功能豐富、現代化的 Web 貪吃蛇遊戲，支援多種遊戲模式、皮膚和高級功能。

## 功能特色

### 🎮 核心遊戲
- **經典無限模式**：蛇在無界的遊戲場景中不斷移動
- **簡單難度設置**：適合所有年齡層的玩家
- **流暢的遊戲體驗**：60 FPS 的高質量渲染

### 📊 計分系統
- **動態計分**：隨著遊戲難度提升而增加分數
- **等級系統**：每吃掉 5 個食物升一級，蛇移動速度加快
- **加速道具**：收集特殊道具獲得臨時加速或無敵效果

### 💾 遊戲數據
- **最高分記錄**：自動保存遊戲最高分
- **排行榜系統**：保存前 20 名玩家的成績
- **玩家統計**：記錄總局數和最高分數

### 🎨 6 種皮膚
1. **經典（綠色）** - 原汁原味的懷舊風格
2. **藍色** - 清爽的藍色主題
3. **紫色** - 神秘的紫色系配色
4. **紅色** - 熱火風格
5. **金色** - 高貴的金色主題
6. **霓虹** - 科技感十足的霓虹效果

### 🛠️ 可調整難度
- **簡單**：初速度 80ms（推薦新手）
- **標準**：初速度 60ms（平衡難度）
- **困難**：初速度 40ms（挑戰難度）
- **極限**：初速度 30ms（終極挑戰）

### 🎁 道具系統
收集不同類型的道具獲得特殊效果：

| 道具 | 效果 | 分數 |
|------|------|------|
| ⭐ 星形 | +50 分，暫時加速 | 50 |
| 🛡️ 盾牌 | 臨時無敵 5 秒 | 0 |
| 🔥 火焰 | 暫時加速 3 秒 | 0 |
| 🍎 食物 | 基礎食物 | 10-20 |

### 🎵 音效系統
- 吃食物音效
- 道具效果音效
- 升級通知音
- 遊戲結束音
- 新紀錄慶祝音

### ⏸️ 暫停功能
- 按空白鍵可隨時暫停/繼續遊戲
- 暫停時可以查看遊戲狀態

## 遊戲操作

### 方向控制
- ⬆️ **上**：按 `W` 或 `↑`
- ⬇️ **下**：按 `S` 或 `↓`
- ⬅️ **左**：按 `A` 或 `←`
- ➡️ **右**：按 `D` 或 `→`

### 遊戲控制
- **空白鍵**：暫停/繼續遊戲
- **按鈕控制**：點擊遊戲界面的按鈕

## 遊戲規則

1. 移動蛇吃掉出現的紅色食物
2. 每吃到一個食物：
   - 蛇身體變長
   - 獲得分數
   - 移動速度逐漸加快
3. **遊戲結束條件**：
   - 撞到遊戲邊界
   - 蛇撞到自己的身體
   - （無敵時期間內則不會結束）

## 計分規則

- **基礎分數**：10 分/個食物
- **等級加成**：`10 × (1 + 等級/10)` 分
- **道具獎勵**：
  - 星形道具：+50 分
  - 盾牌道具：+0 分（特殊效果）
  - 火焰道具：+0 分（特殊效果）

## 遊戲難度

難度等級決定了初始遊戲速度：

| 難度 | 初速度 | 最快速度 | 難度評級 |
|------|--------|---------|---------|
| 簡單 | 80ms | 30ms | ⭐ |
| 標準 | 60ms | 20ms | ⭐⭐ |
| 困難 | 40ms | 20ms | ⭐⭐⭐ |
| 極限 | 30ms | 20ms | ⭐⭐⭐⭐⭐ |

隨著吃掉的食物增加，遊戲速度會自動加快。

## 功能介紹

### 主菜單
- 開始遊戲
- 查看排行榜
- 調整設置
- 了解遊戲規則

### 排行榜
- 顯示前 20 名玩家的成績
- 包含玩家名字、分數和日期
- 金/銀/銅牌圖標標記前三名

### 設置
- 選擇遊戲皮膚
- 切換音效開關
- 調整遊戲難度
- 開啟/關閉道具系統
- 設置玩家名字
- 清除所有遊戲數據

### 關於
- 遊戲規則說明
- 道具系統詳解
- 計分規則說明

## 遊戲結束

遊戲結束後會顯示：
- ✅ 最終分數
- ✅ 蛇的長度
- ✅ 生存時間
- ✅ 達到的速度等級
- 🏆 新紀錄標誌（如果是新紀錄）

可選擇：
- **再來一局**：使用相同設置重新開始
- **返回主菜單**：返回主界面

## 數據保存

所有遊戲數據都保存在瀏覽器的 LocalStorage 中：
- 玩家設置
- 排行榜記錄（前 20 名）
- 最高分數
- 遊戲局數

### 清除數據
在設置頁面點擊「清除所有數據」按鈕可以重置所有遊戲數據。

## 遊戲技巧

1. **保持邊界空間**：盡量不要把自己困在邊角
2. **規劃路線**：提前預測蛇的移動軌跡
3. **收集道具**：當出現特殊道具時要主動去吃
4. **利用無敵**：無敵期間可以大膽冒險
5. **控制速度**：在高速模式下保持冷靜
6. **螺旋走位**：常見的高效率走位方式

## 高分技巧

| 技巧 | 說明 |
|------|------|
| 螺旋攻擊 | 沿著遊戲邊界做螺旋運動 |
| 蛇梯走位 | 來回走位避免被困 |
| 空間管理 | 定期清理身後空間 |
| 道具優先 | 優先去吃高分道具 |
| 速度適應 | 根據難度調整反應速度 |

## 常見問題

**Q: 如何重置高分？**
A: 在設置頁面點擊「清除所有數據」按鈕。

**Q: 能否改變遊戲速度？**
A: 可以的！在設置中選擇不同難度等級。

**Q: 道具會一直出現嗎？**
A: 只有在設置中啟用「道具開關」時才會出現。

**Q: 排行榜怎樣查看？**
A: 在主菜單點擊「排行榜」按鈕查看前 20 名成績。

**Q: 音效可以關閉嗎？**
A: 可以的！在設置中切換「音效」開關。
