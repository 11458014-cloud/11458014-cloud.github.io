# 🎮 個人作品集 (Portfolio)

包含三個高質量的互動式Web應用程式的作品集展示。

## 📁 項目結構

### 🔨 [打地鼠遊戲](./whack-a-mole/)
一個功能豐富的打地鼠遊戲，包含：
- 3種難度模式
- 6種地鼠皮膚
- 雙重排行榜系統
- 等級和連擊系統
- 音效和視覺效果

### 🧮 [工程計算機](./calculator/)
一個多功能的科學計算器，支援：
- 基本四則運算
- 科學函數（三角、對數等）
- 複數運算
- 矩陣操作
- 單位轉換

### 🐍 [貪吃蛇遊戲](./snake/)
經典貪吃蛇遊戲的現代實現：
- 無限增長模式
- 4種難度等級
- 6種蛇皮膚
- 動態難度調整
- 排行榜系統

## 🚀 快速開始

1. **打地鼠**: 進入 `whack-a-mole/` 文件夾，打開 `index.html`
2. **計算機**: 進入 `calculator/` 文件夾，打開 `index.html`
3. **貪吃蛇**: 進入 `snake/` 文件夾，打開 `index.html`

## 💻 技術棧

所有項目均使用：
- **HTML5**：頁面結構
- **CSS3**：響應式設計和動畫
- **JavaScript (ES6+)**：核心邏輯和互動
- **Canvas API**：圖形渲染
- **Web Audio API**：音效系統
- **LocalStorage**：數據持久化
