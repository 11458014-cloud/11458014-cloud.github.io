# 🔨 打地鼠遊戲

一個功能豐富、趣味十足的 Web 打地鼠遊戲，支援多種難度、皮膚和遊戲統計功能。

## 功能特色

### 🎮 核心遊戲
- **經典限時闖關**：在規定時間內盡可能擊中更多地鼠
- **3 種難度模式**：適應不同玩家水平
- **實時分數計算**：動態分數系統，特殊地鼠獎勵高分
- **連擊系統**：連續擊中可獲得額外獎勵

### 📊 計分系統
- **基礎分數**：10 分/次擊中
- **連擊獎勵**：每連擊 +5 分加成
- **黃金地鼠**：30 分（15% 出現率）
- **BOSS 地鼠**：100 分（等級 3+ 時 8% 出現率）
- **等級獎勵**：達到新等級時獲得加成

## 快速開始
1. 用瀏覽器直接打開本文件夾中的 `index.html`
2. 點擊「開始遊戲」按鈕
3. 根據需要選擇難度
4. 點擊出現的地鼠進行擊中
