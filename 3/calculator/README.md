# 🧮 工程計算機

一個功能豐富的 Web 工程計算機，支援多種計算模式和高級功能。

## 功能特色

### 🔢 基本計算模式
- 加、減、乘、除四則運算
- 百分比計算、正負號轉換
- 歷史記錄功能

### 🔬 科學計算模式
- 三角函數：sin, cos, tan
- 對數函數：log、ln
- 冪函數：x²、x³、e^x
- 平方根、階乘、常數 π 和 e

### ℂ 複數運算模式
- 複數加法、減法、乘法、除法
- 複數的模與共軛

### 🔲 矩陣運算模式
- 支援 2×2、3×3 矩陣
- 矩陣轉置、行列式、反矩陣

### 📏 單位轉換模式
- 長度、重量、溫度、速度、面積、體積

## 快速開始
1. 用瀏覽器直接打開本文件夾中的 `index.html`
2. 點擊頂部的模式按鈕切換功能
3. 輸入數值並進行計算
