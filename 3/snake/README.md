# 🐍 貪吃蛇遊戲

一個功能豐富的 Web 貪吃蛇遊戲，支援無限模式、多種難度和排行榜系統。

## 功能特色

### 🎮 核心遊戲
- **無限闖關模式**：一直玩到死
- **4 種難度模式**：簡單到極難
- **動態難度增長**：分數越高速度越快
- **排行榜系統**：記錄前 20 名玩家

### 📊 計分系統
- **基礎食物**：10 分
- **特殊食物**：50 分
- **等級獎勵**：每升級 +100 分

### 🎨 6 種蛇皮膚
1. 經典綠蛇
2. 紅色蛇
3. 藍色蛇
4. 紫色蛇
5. 金色蛇
6. 彩虹蛇

### 🎮 遊戲控制
- 方向鍵或 WASD 控制移動
- 空白鍵暫停/繼續

## 快速開始
1. 用瀏覽器直接打開本文件夾中的 `index.html`
2. 點擊「開始遊戲」按鈕
3. 根據需要選擇難度和蛇皮膚
4. 用方向鍵控制蛇移動，吃掉食物獲得分數
