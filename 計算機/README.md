# 工程計算機 (Engineering Calculator)

一個功能豐富的 Web 工程計算機，支援多種計算模式和高級功能。

## 功能特色

### 🔢 基本計算模式
- 加、減、乘、除四則運算
- 百分比計算
- 正負號轉換
- 歷史記錄功能

### 🔬 科學計算模式
- 三角函數：sin, cos, tan
- 對數函數：log (以 10 為底)、ln (自然對數)
- 冪函數：x²、x³、e^x
- 平方根：√x
- 階乘：n!
- 常數：π (圓周率)、e (自然對數底)
- 支援弧度制和角度制

### ℂ 複數運算模式
- 複數加法、減法、乘法、除法
- 複數的模（絕對值）
- 複數共軛

### 🔲 矩陣運算模式
- 支援 2×2、3×3、2×3、3×2 矩陣
- 矩陣轉置
- 行列式計算（方陣）
- 反矩陣計算（2×2、3×3 方陣）

### 📏 單位轉換模式
- **長度**：米、厘米、毫米、公里、英尺、英寸、碼、海里
- **重量**：克、千克、毫克、噸、磅、盎司
- **溫度**：攝氏度、華氏度、凱爾文
- **速度**：米/秒、公里/小時、英里/小時、節
- **面積**：平方米、平方厘米、平方公里、公頃、平方英尺、平方英寸
- **體積**：立方米、立方厘米、升、毫升、加侖、品脫

## 使用方法

### 開啟計算機
直接在瀏覽器中打開 `index.html` 文件

```bash
# 如果你的計算機在本地伺服器上
http://localhost:8000/5/index.html
```

### 基本操作
1. 點擊數字按鈕輸入數值
2. 選擇運算符（+、−、×、÷）
3. 點擊 `=` 或按 Enter 鍵計算結果
4. 點擊 `C` 清除、`DEL` 刪除最後一位

### 切換模式
點擊頂部的模式按鈕切換不同功能：
- **基本**：基本四則運算
- **科學**：高級數學函數
- **複數**：複數運算
- **矩陣**：矩陣操作
- **單位**：單位轉換

### 科學計算
- 輸入數值後點擊對應的函數按鈕
- 例如：輸入 30，點擊 `sin` 計算 sin(30°)
- 支援鏈式操作

### 複數運算
1. 在實部和虛部輸入框中輸入複數的實部和虛部
2. 選擇運算類型（加、減、乘、除）
3. 輸入第二個複數（會彈出對話框）
4. 結果會顯示在結果框中

### 矩陣運算
1. 選擇矩陣大小（2×2、3×3 等）
2. 點擊「生成空矩陣」生成輸入框
3. 填入矩陣元素
4. 選擇運算類型（轉置、行列式、反矩陣）
5. 結果會顯示在結果框中

### 單位轉換
1. 選擇轉換類型（長度、重量等）
2. 輸入數值
3. 選擇源單位和目標單位
4. 點擊「轉換」按鈕
5. 結果會顯示在結果框中

## 歷史記錄

所有計算結果都會自動保存在歷史記錄中（最多 10 條）。點擊歷史項目可以快速使用之前的結果。

## 已知限制

- 複數運算需要手動輸入第二個複數（會彈出對話框）
- 矩陣運算暫不支援大於 3×3 的矩陣
- 三角函數預設使用弧度制
