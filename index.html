📖 關於作品集

項目概述

這是由 陳稟均 創作的個人作品集， 包含三個功能完整、精心設計的 Web 應用， 展示了現代 Web 開發的各個方面，從計算應用到互動遊戲。

核心特性

  • ✅ 完全響應式設計，適配桌面、平板和手機
  • ✅ 所有數據本地存儲，無需服務器
  • ✅ 現代化的用戶界面和交互設計
  • ✅ 完整的音效和視覺反饋系統
  • ✅ 詳細的遊戲統計和排行榜
  • ✅ 多個皮膚和主題選擇

技術亮點

  • 🎯 使用 Canvas API 實現複雜的圖形渲染
  • 🎯 Web Audio API 合成音效
  • 🎯 複雜的數學計算（複數、矩陣、單位轉換）
  • 🎯 實時遊戲引擎和碰撞檢測
  • 🎯 高效的 DOM 操作和性能優化
