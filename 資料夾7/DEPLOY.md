# 部署指南 - 快速開始

## 第 1 步：準備 Google Sheet

你的試算表應該包含英文單字和中文翻譯。

📋 **必要資訊：**
- Sheet ID: `169b8NtWtT2oJnLRkam-p21sBIGrScyEpfzGVwsUeT2M`
- 工作表名稱: 記下你要使用的工作表名稱（預設為 "工作表1"）

## 第 2 步：在 Google Sheet 中部署 Apps Script

1. 打開你的英文單字試算表
2. 點擊 **工具** → **指令碼編輯器**
3. 此時會打開新的 Google Apps Script 編輯器

## 第 3 步：複製 Code.gs

1. 在指令碼編輯器中，找到 `Code.gs` 標籤
2. 複製此資料夾中 `Code.gs` 文件的全部內容
3. 貼到編輯器中，**替換** 所有現有代碼
4. **修改下列配置：**

```javascript
// 第 3-4 行，修改以下兩個變數

const SPREADSHEET_ID = '169b8NtWtT2oJnLRkam-p21sBIGrScyEpfzGVwsUeT2M';
const SHEET_NAME = '工作表1';  // ← 修改為你的工作表名稱
```

## 第 4 步：建立 HTML 文件

1. 在指令碼編輯器中，點擊 **+ 新增檔案**
2. 選擇 **HTML**
3. 輸入名稱：`TestUI` （重要！不能改）
4. 點擊 **建立**
5. 複製此資料夾中 `TestUI.html` 文件的全部內容
6. 貼到編輯器中

## 第 5 步：測試連接

1. 在 `Code.gs` 編輯器中，找到 `testConnection()` 函數
2. 點擊 **選擇函式** 下拉菜單，選擇 `testConnection`
3. 點擊執行按鈕 (播放按鈕)
4. 點擊 **查看執行記錄** 檢查是否成功讀取單字

如果看到 "取得單字數：" 後面有數字，表示連接成功！

## 第 6 步：部署應用

1. 點擊 **部署** 按鈕
2. 選擇 **新增部署**
3. 設定如下：
   - 📌 類型：選擇 **網頁應用**
   - 🔑 執行者：選擇 **我**
   - 👥 存取權限：選擇 **任何人**
4. 點擊 **部署**
5. 授予權限（Google 會要求確認）
6. 複製部署連結

## 第 7 步：使用測驗

1. 打開部署連結
2. 輸入要進行的測驗題數（1-100）
3. 點擊 **開始測驗**
4. 選擇答案並提交
5. 完成後查看分數和詳細成績單

## 💡 常見問題

### Q: 無法讀取單字資料？
A: 
- 檢查 SHEET_NAME 是否正確拼寫
- 試算表工作表標籤是否存在
- 確保試算表中第 1 和第 2 欄有資料

### Q: 測驗沒有響應？
A:
- 檢查瀏覽器控制台（F12）中是否有錯誤
- 清除瀏覽器快取並重新加載
- 嘗試使用另一個瀏覽器

### Q: 怎樣修改分數滿分為 150？
A: 這個應用設計為滿分 100。如要修改，需修改計算邏輯。

### Q: 能否增加題目難度？
A: 可以。在 `generateOptions()` 中調整選項邏輯，使其更具挑戰性。
