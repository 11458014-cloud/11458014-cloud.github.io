# 🔨 打地鼠遊戲

一個功能豐富、趣味十足的 Web 打地鼠遊戲，支援多種難度、皮膚和遊戲統計功能。

## 功能特色

### 🎮 核心遊戲
- **經典限時闖關**：在規定時間內盡可能擊中更多地鼠
- **3 種難度模式**：適應不同玩家水平
- **實時分數計算**：動態分數系統，特殊地鼠獎勵高分
- **連擊系統**：連續擊中可獲得額外獎勵

### 📊 計分系統
- **基礎分數**：10 分/次擊中
- **連擊獎勵**：每連擊 +5 分加成
- **黃金地鼠**：30 分（15% 出現率）
- **BOSS 地鼠**：100 分（等級 3+ 時 8% 出現率）
- **等級獎勵**：達到新等級時獲得加成

### ⏱️ 遊戲難度
| 難度 | 時間 | 地鼠出現 | 地鼠停留 |
|------|------|---------|---------|
| 簡單 | 60秒 | 1500ms | 1200ms |
| 中等 | 45秒 | 1200ms | 1000ms |
| 困難 | 30秒 | 900ms | 800ms |

### 🎨 6 種地鼠皮膚
1. **經典棕色** - 原汁原味
2. **黃色地鼠** - 可愛外形
3. **粉紅地鼠** - 甜蜜風格
4. **藍色地鼠** - 清爽配色
5. **忍者地鼠** - 酷銳風格
6. **BOSS 地鼠** - 凶悍外形

### 🎁 遊戲特色
- **連擊系統**：連續擊中建立連擊，每 5 連擊時有視覺反饋
- **等級系統**：每擊中 5 個地鼠升一級，等級越高出現的詭異地鼠越多
- **黃金地鼠**：隨機出現，單擊獲得 30 分
- **BOSS 地鼠**：等級 3+ 才能出現，擊中獲得 100 分

### 💾 數據管理
- **排行榜系統**：保存前 20 名玩家成績
- **雙排行榜**：最高分排行榜 + 命中率排行榜
- **遊戲統計**：追蹤最高分、遊戲次數、命中率
- **本地存儲**：所有數據保存在瀏覽器中

### 🔊 音效系統
- 擊中音效
- 遊戲開始音
- 遊戲結束音
- 新紀錄音效
- 暫停/繼續音效

### ⏸️ 暫停功能
- 隨時按空白鍵暫停遊戲
- 暫停菜單可返回主菜單或繼續遊戲

## 遊戲玩法

### 基本規則
1. 點擊出現的地鼠來擊中它
2. 擊中獲得分數，連續擊中增加連擊數
3. 在規定時間內盡可能多地擊中地鼠
4. 時間結束遊戲自動結束
5. 避免浪費 20 次可用擊中次數

### 計分規則
```
基礎分數 = 10 分（每次擊中）
連擊加成 = 當前連擊數 × 5 分
黃金地鼠 = 30 分
BOSS 地鼠 = 100 分
等級加成 = 50 分（每升級時）
```

### 等級系統
- 每擊中 5 個地鼠升一級
- 等級越高，BOSS 地鼠出現率越高
- 等級 3+ 才能出現 BOSS 地鼠

## 遊戲技巧

1. **眼睛準備**：提前準備好快速點擊，地鼠出現時間有限
2. **專註屏幕**：將注意力集中在遊戲板中央
3. **保持連擊**：連續擊中會獲得額外獎勵
4. **追蹤特殊地鼠**：優先擊中黃金地鼠和 BOSS 地鼠
5. **不要浪費機會**：每個機會都很寶貴
6. **難度選擇**：根據自己的反應速度選擇合適的難度

## 高分技巧

| 策略 | 說明 |
|------|------|
| 提前移動 | 提前將鼠標移向常出現的地鼠孔洞 |
| 連擊優先 | 優先保持連擊，而不是追求絕對分數 |
| 特殊追蹤 | 留意黃金和 BOSS 地鼠的位置 |
| 時間管理 | 在時間充足時放鬆，時間不足時加快速度 |
| 練習習慣 | 反覆練習同一難度來建立肌肉記憶 |

## 數據保存

所有遊戲數據都保存在瀏覽器的 LocalStorage 中：
- 玩家設置（名字、難度、皮膚等）
- 排行榜記錄（前 20 名）
- 最高分數
- 遊戲局數

### 清除數據
在設置頁面點擊「清除數據」按鈕可以重置所有遊戲數據。

## 常見問題

**Q: 如何重置高分？**
A: 在設置頁面點擊「清除數據」按鈕。

**Q: 能否改變遊戲難度？**
A: 可以的！在設置中選擇簡單、中等或困難難度。

**Q: 怎樣獲得更高的分數？**
A: 保持連擊、追蹤特殊地鼠、在更高難度中遊戲都能獲得更高分數。

**Q: 排行榜怎樣查看？**
A: 在主菜單點擊「排行榜」按鈕查看前 20 名成績。

**Q: 音效可以關閉嗎？**
A: 可以的！在設置中切換「音效」開關。

**Q: 怎樣設置玩家名字？**
A: 在設置中「玩家名字」欄輸入你的昵稱，然後點擊「保存設置」。
